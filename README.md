# C#
## Curso Udemy

Curso Completo de C# com orientação a objeto. Além de cobrir a teoria de forma clara e objetiva, o curso possui muito enfoque prático. Cada seção do curso possui exercícios resolvidos e exercícios propostos para você treinar. E nós vamos ainda fazer projetos neste curso, os quais vão te permitir colocar tudo que você aprendeu em prática, e vai te permitir compreender como os tópicos se unem para a construção de projetos reais.
